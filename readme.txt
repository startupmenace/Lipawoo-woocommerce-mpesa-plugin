Accept M-Pesa STK Push payments in WooCommerce using the Safaricom Daraja API. Includes 5 free production transactions.

== Description ==

LipaWoo Lite integrates M-Pesa STK Push into your WooCommerce store. Test with sandbox credentials for free, or use production credentials for up to 5 real transactions — no credit card required.

= Features =
* M-Pesa STK Push via Safaricom Daraja API
* Real-time payment confirmation via STK Query polling
* Supports both Paybill and Till (Buy Goods)
* Transaction log dashboard in WooCommerce admin
* Clean, mobile-friendly payment UI
* Unlimited sandbox testing + 5 free production transactions

= Lite vs Pro =
LipaWoo Lite includes 5 free production transactions. Upgrade to LipaWoo Pro at https://mpesa-woocommerce.wasmer.app for unlimited production payments, priority support, and premium features.

== Installation ==
1. Upload the plugin to /wp-content/plugins/
2. Activate through Plugins menu
3. Go to WooCommerce > Settings > Payments > M-Pesa LipaWoo Lite
4. Enter your Safaricom sandbox credentials

== Frequently Asked Questions ==

= Can I take real payments with this plugin? =
Yes! LipaWoo Lite includes 5 free production transactions. Upgrade to Pro for unlimited production payments.

= Where do I get credentials? =
Sandbox credentials: developer.safaricom.co.ke — create an account, make an app, and add the Lipa Na M-Pesa Online API product. Production credentials require a Safaricom business account.
